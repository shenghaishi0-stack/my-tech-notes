# 我的博客与技术笔记仓库

这是我的个人博客与技术笔记仓库，用来记录学习过程、项目实践和折腾经验。  
内容主要围绕 **AI 应用、本地部署、智能家居、嵌入式开发** 等方向展开，同时也会整理一些实用工具、开发方案和踩坑记录。

## 仓库内容

本仓库主要包含以下几个部分：

- **博客文章**
  - 记录日常学习笔记、项目总结、部署教程与经验复盘
  - 包括从零搭建、本地运行、问题排查、优化实践等内容

- **AI 应用**
  - 本地大模型与智能体相关实践
  - OpenClaw、多智能体、工作流自动化、RAG 知识库等方向
  - AI 在效率工具、内容创作、智能控制中的应用探索

- **智能家居**
  - Home Assistant 相关配置与笔记
  - 智能设备接入、自动化场景设计、面板美化、局域网服务整合
  - 打造更智能、更易用的个人家庭控制中心

- **嵌入式开发**
  - Arduino、ESP32、FreeRTOS 等相关学习与项目实践
  - 传感器接入、设备控制、通信协议、物联网应用
  - 面向比赛、课程设计和实际项目的实现记录

- **项目展示**
  - 个人做过的软硬件项目与原型
  - 包括智能设备、AI 小工具、自动化系统、技术演示项目等
